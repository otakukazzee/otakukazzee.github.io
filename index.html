Tentang Saya

Saya adalah seorang programmer yang memiliki ketertarikan mendalam terhadap dunia teknologi dan anime. Dengan latar belakang dalam pengembangan perangkat lunak, saya selalu berusaha untuk mengasah keterampilan saya dalam berbagai bahasa pemrograman dan alat pengembangan.

Minat dalam Anime:

  • Kreativitas dan Cerita: Anime memberikan inspirasi bagi saya dalam menciptakan cerita dan karakter yang menarik. Saya sering mengambil elemen dari anime untuk diterapkan dalam proyek pengembangan saya.
  • Komunitas: Saya aktif terlibat dalam komunitas anime, berbagi rekomendasi dan berdiskusi tentang berbagai judul. Ini membantu saya tetap terhubung dengan orang-orang yang memiliki minat yang sama.

Dunia Teknologi:

  • Inovasi: Saya selalu mengikuti perkembangan terbaru dalam teknologi, termasuk kecerdasan buatan, pengembangan web, dan aplikasi mobile. Saya percaya bahwa teknologi dapat digunakan untuk menciptakan solusi yang bermanfaat bagi masyarakat.
  • Proyek Pribadi: Saya sering mengerjakan proyek pribadi yang menggabungkan minat saya dalam anime dan teknologi, seperti aplikasi yang merekomendasikan anime berdasarkan preferensi pengguna.

Dengan kombinasi keterampilan teknis dan minat yang kuat terhadap anime, saya berusaha untuk menciptakan karya yang tidak hanya fungsional tetapi juga menarik secara visual dan naratif.

Portofolio

Simulasi Sistem Operasi dengan Python

Proyek ini menggunakan Python dan Tkinter untuk membuat simulasi sistem operasi. Saya mengembangkan antarmuka pengguna yang interaktif untuk menggambarkan proses manajemen memori dan penjadwalan proses. Ini membantu saya memahami konsep dasar sistem operasi dan bagaimana mereka berfungsi.

Aplikasi Mobile dengan Flutter

Proyek ini menggunakan Flutter sebagai framework untuk mengembangkan aplikasi mobile yang responsif dan menarik. Flutter memungkinkan saya untuk membangun antarmuka pengguna yang indah dan performa tinggi dengan satu basis kode untuk iOS dan Android. Saya sangat menyukai kemudahan dan fleksibilitas yang ditawarkan oleh Flutter dalam pengembangan aplikasi.
